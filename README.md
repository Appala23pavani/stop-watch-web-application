# ⏱ Stopwatch Web Application

## 📖 Project Overview

This project is a responsive Stopwatch Web Application developed as **Task 02** during my **Web Development Internship at SkillCraft Technology**.

The application allows users to accurately measure time with essential stopwatch features such as Start, Pause, Reset, and Lap Time recording. It is built using HTML, CSS, and JavaScript with a clean and user-friendly interface.

## ✨ Features

- ▶️ Start the stopwatch
- ⏸️ Pause and resume the timer
- 🔄 Reset the stopwatch
- 📝 Record multiple lap times
- 📱 Responsive design for desktop and mobile devices
- 🎨 Simple and modern user interface

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
